# generator-hydraweb

To install generator-hydraweb from npm, run:

```bash
npm install -g generator-hydraweb
```

Go to your project's folder:

```bash
cd my/new/project
```

Finally, initiate the generator:

```bash
yo hydraweb
```


Prereqs and installation requirements
=====================================

install [yeoman](http://yeoman.io/).
```shell
npm install -g yo
```
